# 12 · 组卷与导出版式引擎

## 1. 组卷流程

```text
选目标 → 生成初稿 → 换题微调 → 审校 → 导出
```

普通模式不要求教师填写复杂细目表。

## 2. 题目推荐排序

默认综合推荐分：

```text
知识点匹配度
+ 题型匹配度
+ 难度匹配度
+ 质量分
+ 历史区分度
+ 答案解析完整度
+ 地区考点相关度
- 近期重复惩罚
- 相似题重复惩罚
- 低质量惩罚
```

教师可切换排序：综合推荐、最匹配知识点、最近未使用、高质量优先、真题优先、适合讲评优先。

## 3. 推荐理由

每道推荐题显示简短理由：

```text
命中“浮力计算”主知识点；历史得分率 0.62；近 90 天未使用；适合中等题。
```

## 4. 导出版本

- 学生版。
- 教师版。
- 答案版。
- 讲评版。
- A/B/C 分层练习版。

## 5. 模板层级

| 模板 | 内容 |
|---|---|
| 页面模板 | A4/A3、页边距、页眉页脚 |
| 试卷模板 | 标题、学校、班级、姓名、时间 |
| 大题模板 | 选择、填空、实验、计算 |
| 选项模板 | 一行四项、两行两项、每项一行 |
| 答案模板 | 简明答案/详细解析 |
| 答题空间模板 | 空白、横线、方框 |

## 6. 选项排版规则

```text
选项短 → 一行四项
选项中等 → 两行两项
选项长/含公式/图片 → 每项一行
```

## 7. 导出前审校

检查：总分、题号、答案解析、知识点覆盖、重复题、近期使用、图片缺失、公式损坏、表格溢出、题干条件不足。

## 8. 导出回归测试

每次修改导出引擎，要用黄金样本生成 docx/pdf/图片并检查：题号、公式、图片、表格、分页、WPS 打开兼容性。
