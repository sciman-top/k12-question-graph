# 10 · 人工确认、反馈学习与系统演进

## 1. 原则

人工修改自动记录为结构化反馈，教师不额外填写复杂表格。

## 2. 自动反馈流程

```text
教师修改 AI 结果
→ 系统自动 diff 修改前后
→ 生成 FeedbackEvent
→ 可选让教师点一个原因标签
→ 进入反馈样本库
→ 定期分析高频错误
→ 生成候选规则/prompt 优化
→ 黄金样本集评测
→ 人工批准发布
→ 可回滚
```

## 3. FeedbackEvent 字段

```text
id
task_type
entity_type
entity_id
field_key
before_value
after_value
ai_confidence
reason_tag
teacher_id
created_at
prompt_version
schema_version
model
accepted_for_eval
```

## 4. 应纳入反馈样本的内容

| 类型 | 样本 |
|---|---|
| 切题 | 边界、小问、跨页题 |
| 题图 | 共用题图、题图归属 |
| OCR | 错字、漏字、乱码、单位 |
| 公式 | LaTeX 修正、OMML 转换失败 |
| 表格 | 合并单元格、表头、结构 |
| 知识点 | 主知识点、副知识点、小问知识点 |
| 映射 | 教材章节、课标、地区考点 |
| 答案解析 | 答案提取、校验、解析补全 |
| 难度质量 | AI 预估与教师/实测差异 |
| 组卷 | 教师替换题、保留题、拒绝题 |
| 成绩导入 | Excel 字段映射修正 |
| 报告 | 教师修改过的分析文字 |

## 5. 教师快速标签

支持单选、多选、评分、布尔、数值、文本、层级选择、关系字段。

示例：

| 类型 | 标签 |
|---|---|
| 单选 | 使用状态：推荐/可用/需优化/暂停/弃用 |
| 多选 | 适用场景：新授/复习/讲评/分层 |
| 评分 | 教学适用性 1-5 星 |
| 布尔 | 是否适合正式考试 |
| 文本 | 教师备注 |

## 6. 减少评分差异

主观评价必须有量规和锚定样例。题目质量等级 A/B/C/D 要定义明确，校本题库正式题目可采用多人评价和备课组裁定。
