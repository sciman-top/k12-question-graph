# P0 Rollback · 2026-05-02

## Code Rollback

Inspect changes:

```powershell
git status --short
git diff --stat
```

Revert selected files:

```powershell
git restore --source=HEAD -- <path>
```

Do not use `git reset --hard` unless explicitly approved.

## Database Rollback

Revert EF migration:

```powershell
$env:KQG_CONNECTION_STRING='Host=127.0.0.1;Port=5432;Database=k12_question_graph;Username=postgres;Password=<local-password>'
dotnet tool run dotnet-ef database update 0 --project apps\api\K12QuestionGraph.Api.csproj --startup-project apps\api\K12QuestionGraph.Api.csproj
```

Drop local database:

```powershell
$env:PGPASSWORD='<local-password>'
& 'C:\Program Files\PostgreSQL\17\bin\dropdb.exe' -h 127.0.0.1 -U postgres k12_question_graph
```


## Host Dependency Rollback

Uninstall PostgreSQL 17:

```powershell
winget uninstall --id PostgreSQL.PostgreSQL.17
```

Only do this if no other local project depends on the service `postgresql-x64-17`.
