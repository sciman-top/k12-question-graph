# 15 · 缓存、存储与文件生命周期

## 1. 会产生的缓存

| 类型 | 示例 | 策略 |
|---|---|---|
| OCR 临时文件 | 页图、切图、中间文本 | 定期清理 |
| 文档解析缓存 | 解析 JSON、页面块 | 短期保留 |
| 缩略图 | 题卡预览图 | 可重建，空间不足可清理 |
| 导出缓存 | 临时 Word/PDF/图片 | 按时间清理 |
| AI 响应缓存 | 同输入同 prompt 结果 | 可长期保留部分 |
| Embedding | 向量索引 | 可重建但不随意删 |
| 日志 | 操作/错误/AI 成本 | 分级保留 |
| 失败任务现场 | 异常上下文 | 30 天左右 |

## 2. 原始文件是否长期保存

不必全部永久保存，但不能默认粗暴删除。

策略：

```text
结构化题目长期保存
题目级证据长期保存
普通原始文件可压缩、归档、保留期后清理
真题/校本正式卷/教材/课标长期保存
```

删除前必须满足：已入库、已确认、无待审核、非关键资料、已备份、已保留题目截图和素材。

## 3. 文件状态

```text
已上传
→ 已解析
→ 已入库
→ 已确认
→ 已压缩归档
→ 待清理
→ 回收站
→ 永久删除
```

## 4. 默认保留策略

| 类型 | 默认 |
|---|---|
| 已上传未处理 | 30 天提醒，90 天可入回收站 |
| 已入库普通资料 | 原件保留 90-180 天后可归档/删除 |
| 真题/校本正式卷/教材课标 | 长期保留 |
| 临时导出 | 30-90 天 |
| OCR 中间文件 | 7-30 天 |
| 回收站 | 30 天 |

## 5. 文件资产中心

管理员可查看：未处理资料、重复资料、大文件、待清理资料、缓存占用、备份占用、可安全清理容量。

## 6. 清理前影响分析

删除前显示：将删除哪些文件，影响哪些题目，是否已备份，是否可从回收站恢复。
