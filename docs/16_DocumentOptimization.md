# 16 · 文档压缩、优化与清晰化

## 1. 目标

控制存储体积，同时不损坏物理题图、公式、表格和 OCR 可读性。

## 2. Document Optimization Worker

处理：

```text
docx/xlsx/pptx 嵌入图片压缩
PDF 安全优化
图片裁边、纠偏、去阴影、去噪、压缩
缩略图生成
重复文件检测
优化前后质量检查
```

## 3. 三档优化

| 档位 | 用途 | 风险 |
|---|---|---|
| 安全优化 | 默认，正式资料 | 不降低分辨率，主要清理冗余 |
| 平衡压缩 | 普通练习资料 | 适度降采样，保持可读 |
| 强压缩 | 临时资料 | 默认关闭，可能损失细节 |

## 4. docx 优化

docx 本身是 ZIP + XML，二次压缩收益有限。真正有效的是：压缩内嵌图片、删除未引用媒体、清理缩略图和元数据、重新打包。

## 5. PDF 优化

使用 OCRmyPDF/Ghostscript/qpdf 的适配层。默认安全优化；强压缩必须由管理员或教师明确选择。

## 6. 图片优化

保存三层：

```text
original_image
optimized_image
thumbnail_image
```

重要题图尽量保留原图；普通资料可保留优化图和缩略图。

## 7. 质量检查

优化后检查：页数、图片数量、公式可读性、OCR 识别率、题图清晰度、Word/WPS 打开兼容性。
